About

MAYA (My Archieve Your Archieve) is used to make web clients that can interact with aarhus stadsarkivs API service. Aarhus stadsarkiv contains a lot of public archives and the API service is used to search and retrieve data from the archives.

This page contains some information about the project and how to use it. The project is open source and the code is available on GitHub. The project is developed by Aarhus Stadsarkiv and is used in the aarhusarkivet.dk website.
